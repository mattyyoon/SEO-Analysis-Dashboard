I graduated this past fall semester with a degree in math and I'd like to use my analytical background in a more application-based setting.
Although I haven't created dashboards or used queries during school, this is my first project utilizing what I've learned so far post-grad.

The goal of this project was not only to familiarize myself with Tableau and SQL but to create a project that allowed me to create meaningful insights.
I used a sample dataset from the Google Merchandise Store which emulates data typical from an ecommerce website due to inspiration from a LinkedIn Job post.

This project analyzes organic search performance using Google Search Console data to identify high-impact SEO optimization opportunities.

Using Tableau, I built an interactive dashboard that visualizes the relationship between search visibility (Impressions) and engagement (Click-Through Rate) at the page level. 
A scatter plot was created to segment pages based on above- and below-median performance, allowing for the identification of high-visibility pages with underperforming CTR — key candidates 
for metadata and snippet optimization.

To prioritize scalable impact, I filtered the analysis to focus on top traffic-driving pages and applied median-based thresholds to ensure the segmentation was data-driven rather than arbitrary. 
Pages were cleaned and reformatted using calculated fields (regex extraction) to improve readability and presentation quality.

Additionally, I developed dynamic tooltip logic that generates performance insights directly within the visualization. This allows stakeholders to quickly understand not only performance metrics 
(Clicks, Impressions, CTR), but also recommended optimization actions based on page performance patterns.

The final dashboard highlights:

High-impact pages driving organic traffic
Pages with strong visibility but low engagement
Engagement-efficient pages with expansion potential
Actionable SEO optimization targets
